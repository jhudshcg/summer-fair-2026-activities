Tips for New Programmers

If you're new to these coding archipelagos, give these tips a go to help you on your way.

  • Programs run in order, one instruction or step (represented by a block) at a time.
  • Use the repeat blocks to repeat steps. Each repeat block needs a condition (or count range) to run and control the number of repetitions.
  • Variables are just named, stored values that you can use to keep track of values through a program. The value of a variable can change as the program runs.
  • A condition like "pass from 2 to length(numbers)" means there is a variable 'pass' that starts at 2 and counts up by 1 each repetition, until it reaches the length of the numbers list (in this case 6).
  • Each number in the numbers list has a position, or index, so to access the number at index 2 (the 3rd position from the left), you would use numbers[2]. In the blocks below, both 'numbers' and 'index' are examples of variables.
  • Use the choice blocks to check if something is true, e.g. whether one number is bigger than the next.
  • Use the swap block to swap two numbers at the given positions around in the list.
  • For a program to work as intended blocks may need to be placed inside other blocks, e.g. a choice and repeat inside another repeat.
  • Check your thinking by clicking "Check program" to see if your program matches the bubble sort algorithm.
