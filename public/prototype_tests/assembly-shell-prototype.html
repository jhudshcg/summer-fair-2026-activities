Assembly Shell Prototype

Scratch-only markup for validating the open-right container shape before porting it back into the live assembly renderer.

Prototype Intent

The outer repeat and choice shells below are standalone markup. They are meant to answer one question only: does the container itself read as a boxy letter C with a closed top, open-right middle, and closed bottom?

Repeat
Condition index from 0 to length(numbers) - pass:
Repeat
Condition pass from 2 to length(numbers):
Choice If
Condition numbers[index] > numbers[index + 1]
Step swap(numbers, index, index + 1)
Step set swapped to true
Step move to next pair
Step finish this pass
