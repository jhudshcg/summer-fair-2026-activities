Tips for New Programmers

If you're new to these coding archipelagos, give these tips a go to help you on your way.

  • Programs run in order, one instruction or step (represented by a block) at a time.
  • Each move forward block sends the explorer one space along the trail, so the number of forward moves matters.
  • Use the repeat blocks when a pattern happens more than once. In this maze that might mean repeating a short route for a set count, or repeating until the explorer reaches the finish.
  • Use the choice blocks to react to the maze. Here they can be used to check whether there is a path on the left or right before deciding to turn.
  • The turn blocks change which direction the explorer is facing, so the same next move can lead somewhere different after a turn left or turn right.
  • For a program to work as intended blocks may need to be placed inside other blocks, for example a choice block inside a repeat block.
  • There can be multiple working solutions, and some good solutions may not need every block from the palette.
  • Check your thinking by clicking "Check solution" to see whether your program guides the explorer all the way to the finish.
