# Summer Fair 2026 Activities

A set of mobile first web page activities for the SHCG Summer Fair 2026 event, to promote the Southampton College Digital offer.

The activities are designed to be engaging and interactive, providing attendees with a fun and educational experience, relating to computational thinking, problem-solving and computer programming.

The pages are mobile first, meaning they are optimized for viewing and interaction on mobile devices, ensuring accessibility for all attendees.

Access can be given with a QR code, which can be scanned by attendees to access the activities directly on their mobile devices.

There are 4 activities, with a locked bonus 5th activity, available only to those who complete the first 4 activities. The activities are designed to be completed in a short amount of time, making them suitable for a fair environment.

Theme: tropical (for some reason).
